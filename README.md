## Quick Start

### Local Development
1. Clone the repository
2. Open `index.html` in a web browser
3. Or use a local server:
   ```bash
   npm install
   npm run dev
   ```

### Deployment on Vercel
1. Push code to GitHub repository
2. Connect repository to Vercel
3. Deploy automatically
