# 1602 Lucky Draw - GitHub Pages Deployment

## 🚀 GitHub Pages 部署指南

### 自动部署
项目已配置GitHub Actions自动部署到GitHub Pages。每次推送到main分支时会自动触发部署。

### 手动启用GitHub Pages
1. 进入GitHub仓库设置页面
2. 找到"Pages"选项
3. 在"Source"中选择"GitHub Actions"
4. 保存设置
